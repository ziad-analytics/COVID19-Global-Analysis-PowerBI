# COVID-19 Global Analysis — Power BI

## Overview
Interactive Power BI dashboard analyzing global COVID-19 pandemic data 
across WHO regions and 180+ countries.

## Key Metrics
- 16M Total Confirmed Cases
- 9M Total Recovered
- 229K New Cases
- 176K+ Country-level records

## Features
- Global map visualization by WHO region
- Death and recovery trends across regions
- Weekly percentage increase tracking
- Country-level breakdown table
- Confirmed vs
